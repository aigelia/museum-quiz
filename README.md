# Бот для викторин

Бот для VK и Telegram, который умеет проводить викторины по загруженным в него вопросам и ответам. Архитектура подразумевает получение вопросов и ответов из файлов определенного типа, примеры можно найти в каталоге `quiz-questions` внутри репозитория.

## Установка

Клонируйте репозиторий и создайте виртуальное окружение с `Python 3.12.3`. Проект использует пакетный менеджер `uv` для хранения зависимостей. Установите его и зависимости:

```shell
pip install uv
uv sync
```

Создайте файл `.env` и добавьте в него переменные окружения:

- `TG_TOKEN` - токен бота TG;
- `VK_TOKEN` - токен группы VK;
- `REDIS_HOST` - хост базы данных Redis;
- `REDIS_PORT`- порт базы данных Redis;
- `REDIS_PASSWORD` - пароль для базы данных.
